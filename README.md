# WebOS

A mini browser-based operating system built with HTML, CSS, and JavaScript.

## What it does
WebOS is a simple desktop-style interface that runs in the browser. It includes:

- Welcome window (startup screen)
- Notes app (type and edit text live)
- Search app (type and see results instantly)
- Settings app (toggle space theme background)
- Live clock in the top bar
- Dock with clickable app icons
- Mac-style close buttons for windows

## Features
- Click icons in the dock to open apps
- Type directly inside the Notes app
- Use the Search app to type and see results
- Switch between background themes in Settings
- Close windows using the close button

## Built With
- HTML
- CSS
- JavaScript
- Font Awesome

## Theme
The default theme is a space-style desktop interface inspired by modern operating systems with a glass-like design.

## How to run
Open `index.html` in any browser.

Or deploy using GitHub Pages.
